🎓 AI-Based Student Placement Prediction System
🚀 Live Demo
👉 https://student-placement-ml-kuldeep-lakhera.streamlit.app/
📌 Project Overview
This project predicts whether a student is likely to get placed based on academic performance and skill metrics.
The system compares multiple machine learning models and selects the best-performing one.
🧠 Machine Learning Models Used
Logistic Regression
Decision Tree
Random Forest
📊 Features
Model accuracy comparison
Confusion matrix visualization
Placement probability prediction
Multi-model evaluation
Live web deployment
🛠 Tech Stack
Python
Scikit-learn
Pandas
Streamlit
Matplotlib
Seaborn
📈 Model Evaluation
Models are compared using accuracy scores.
The best-performing model is automatically selected for predictions.
🌍 Deployment
Deployed on Streamlit Cloud.
